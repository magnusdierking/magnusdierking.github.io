  • Incremental Strategy
    • Determine cyclic path through all agents
    • Step through cycle, each agent reveices from previous and updates according to own cost function.
    • Drawbacks
      • All agents need update after each cycle
      • Cycle hard to determine in large graphs (NP)
      • Limited cooperation, as each agent sends and receives only once per iteration
      • Sensitive to link or agent failures (new path needed)
