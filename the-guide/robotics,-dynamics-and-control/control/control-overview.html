Control Overview

In a Nutshell

Field of applied mathematics dealing with control of dynamical systems.

Basic Lingo

Control in Joint Space

Control based on error between actual and desired joint positions and velocities.

Control in Joint Space

Control based on desired end-effector state in cartesian coordinates , requiring solving inverse kinematics to map them into joint space.

Open-Loop Control

Control strategy where the controller sends control commands to the system without using feedback from the system's output (i.e., without measuring the system's current state). In other words, the control action is predetermined and not adjusted based on the system's actual performance.

Closed-Loop Control

Uses feedback from the system's output to adjust the input in real-time, allowing the system to self-correct its behavior and maintain the desired performance.

Basic Control Approaches

  • Position Control: Directly regulates the position of the robot. Commonly uses basic PD or PID controllers.
    • Governs the robot's joint positions by minimizing the error between the current and desired positions.
    • Often used in tasks that require precise positioning.
  • Velocity Control: Focuses on controlling the velocity of the robot. Often used in high-speed motion tasks.
    • Involves controlling the joint velocities by minimizing the error between the current and desired velocities.
    • Useful in tasks that require fast movements or when position tracking errors need to be minimized dynamically.
  • Torque Control: Directly controls the torques applied to the robot's joints. Often used in force-based tasks.
    • Allows direct manipulation of forces and is typically used when compliance or interaction with an environment is required.
